# SentinelVibe 🛡️ 
[🚀 View Live Demo](https://pulse-sentiment-engine.vercel.app)
## | AI-Powered Brand Sentiment Dashboard

SentinelVibe is a high-fidelity sentiment analysis dashboard designed to monitor brand reputation in real-time. It bridges the gap between qualitative feedback (social media mentions) and quantitative data (analytics) using a custom heuristic analysis engine.


## 🚀 The Real-World Problem
In 2026, information moves faster than ever. Businesses struggle to keep track of their "vibe" across multiple platforms. SentinelVibe solves this by:
1. **Aggregating** mentions from various sources (Simulated).
2. **Analyzing** text for sentiment (Positive, Neutral, Critical).
3. **Filtering** data so managers can focus on critical alerts immediately.

## 🛠️ Tech Stack
- **Frontend:** HTML5 & Tailwind CSS (Utility-first styling).
- **Logic:** Vanilla JavaScript (ES6+).
- **Design:** Modern "Glassmorphism" UI with dark-mode defaults.
- **Workflow:** Vibe Coding (Orchestrating intent through AI-assisted development).

## ✨ Key Features
- **Heuristic Sentiment Engine:** Analyzes user input and simulated data streams for keywords to categorize "vibe" scores.
- **Interactive Injection Zone:** Users can manually "inject" mentions to test the analysis logic in real-time.
- **Dynamic Stats:** A live-updating statistics grid that recalculates sentiment scores without page refreshes.
- **Fluid UI:** Responsive design with staggered CSS animations for an "agentic" feel.
